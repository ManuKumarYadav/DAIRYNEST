# DairyNest

DairyNest is a full-stack dairy management system built with the MERN stack. It streamlines dairy operations by managing farmers, milk collection, inventory, products, orders, staff, deliveries, and online payments.

##  Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT
- Razorpay
